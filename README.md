# DevelopersHub-Internship-Portfolio-Phase-II
# Overview

This repository contains the second phase of my Data Analytics Internship at DevelopersHub, where I successfully completed five end-to-end analytics projects covering machine learning, predictive analytics, customer segmentation, risk assessment, time-series forecasting, and business intelligence dashboard development.

Each project followed a structured data analytics workflow, including data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, model development, evaluation, visualization, explainability, and business-driven recommendations.

The projects were designed to solve real-world business challenges across marketing, banking, retail, customer analytics, and energy consumption domains while demonstrating the practical application of data science and machine learning techniques.

# Projects Included

# Task 1: Bank Marketing Campaign Prediction

Developed classification models to predict customer subscription to term deposit marketing campaigns. The project involved feature encoding, model training using Logistic Regression and Random Forest, performance evaluation using Confusion Matrix, F1-Score, ROC Curve, and SHAP explainability to interpret individual predictions and feature importance.

Key Skills: 

* Classification Modeling

* Feature Engineering

* SHAP Explainability

* Model Evaluation

* Business Analytics

# Task 2: Customer Segmentation using K-Means Clustering

Applied unsupervised machine learning techniques to segment mall customers into meaningful groups based on demographic characteristics and spending behavior. PCA was used for cluster visualization, and tailored marketing strategies were developed for each identified customer segment.

Key Skills: 

* Customer Segmentation

* Clustering

* K-Means

* PCA

* Marketing Analytics

# Task 3: Household Power Consumption Forecasting

Developed forecasting models using ARIMA, Prophet, and XGBoost to predict future household energy consumption. The project included time-series preprocessing, feature engineering, model comparison, forecast evaluation, and visualization of actual versus predicted energy usage.

Key Skills: 

* Time-Series Analysis

* Forecasting

* ARIMA

* Prophet

* XGBoost

* Energy Analytics

# Task 4: Credit Risk Assessment and Business Cost Optimization

Built predictive models to assess loan default risk using customer demographic and financial information. Logistic Regression and CatBoost models were trained and evaluated, while business cost analysis was used to optimize decision thresholds and minimize the financial impact of false positives and false negatives.

Key Skills: 

* Credit Risk Modeling

* Cost-Sensitive Learning

* Threshold Optimization

* Classification

* Business Decision Analytics

# Task 5: Global Superstore Sales Analysis and Interactive Dashboard

Performed sales and profitability analysis using retail transaction data and developed an interactive Streamlit dashboard featuring dynamic filters for Region, Category, and Sub-Category. The dashboard provides real-time insights into sales performance, profit trends, and top-performing customers.

Key Skills: 

* Business Intelligence

* Dashboard Development

* Streamlit

* KPI Reporting

* Data Visualization.
