# Bank Marketing Campaign Prediction using Machine Learning

# Project Overview
This project focuses on predicting whether a customer will subscribe to a term deposit based on demographic information, financial characteristics, and previous marketing campaign interactions.

The analysis involved data preprocessing, feature encoding, model training, performance evaluation, and explainable AI techniques to understand the key drivers influencing customer subscription decisions.

# Key Activities
* Data Cleaning and Preprocessing 
* Categorical Feature Encoding
* Exploratory Data Analysis (EDA)
* Logistic Regression Model Development
* Random Forest Model Development

* Model Evaluation using: 
  * Confusion Matrix 
  * F1-Score 
  * ROC Curve 
  * ROC-AUC Score 
  * SHAP-based Model Explainability
* Business Insights and Recommendations

# Key Findings
* Contact duration was identified as the most influential predictor of customer subscription.
* Previous successful campaign outcomes significantly increased the likelihood of subscription.
* Random Forest outperformed Logistic Regression in predictive performance.
* SHAP analysis provided transparency into model predictions and highlighted the importance of customer engagement history.

# Business Value
The solution enables organizations to improve marketing efficiency by identifying customers with a higher likelihood of subscribing to financial products, thereby optimizing campaign targeting and resource allocation.
