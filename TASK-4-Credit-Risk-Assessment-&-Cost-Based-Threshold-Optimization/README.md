# Credit Risk Assessment and Business Cost Optimization

# Project Overview
This project focuses on predicting loan default risk using machine learning techniques and optimizing lending decisions through business cost analysis. 
The goal was to identify high-risk borrowers, evaluate model performance, and determine the optimal decision threshold that minimizes financial losses associated with lending.

# Key Activities
* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering and Encoding
* Logistic Regression Model Development
* CatBoost Model Development
* Model Evaluation using:
  * Confusion Matrix
  * Classification Report
  * F1-Score
  * ROC Curve
  * ROC-AUC
* Business Cost Analysis
* Threshold Optimization
* Risk-Based Recommendations

# Key Findings
* Previous credit behavior and loan-related features significantly influence default risk.
* CatBoost outperformed Logistic Regression in predictive performance.
* Threshold optimization demonstrated that business costs can be reduced by balancing false positives and false negatives.
* Risk-based lending strategies can improve portfolio performance and reduce default-related losses.

# Business Value
The solution provides a data-driven framework for credit risk assessment, enabling financial institutions to improve lending decisions, reduce financial exposure, and optimize risk management processes.
