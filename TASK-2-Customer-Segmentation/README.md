# Customer Segmentation using K-Means Clustering

# Project Overview
This project applies unsupervised machine learning techniques to segment mall customers into meaningful groups based on demographic characteristics and spending behavior.

The objective was to uncover customer segments that can support personalized marketing strategies and improve customer relationship management.

# Key Activities
* Data Cleaning and Preparation
* Exploratory Data Analysis (EDA)
* Feature Scaling
* K-Means Clustering
* Elbow Method for Optimal Cluster Selection
* PCA-based Cluster Visualization
* Cluster Interpretation
* Marketing Strategy Development

# Key Findings
* Five distinct customer segments were identified.
* Customers exhibited varying spending patterns despite similar income levels.
* PCA visualization confirmed clear separation among customer groups.
* High-income customers were not always the highest spenders, highlighting the need for targeted engagement strategies.

# Business Value
Customer segmentation enables businesses to design personalized marketing campaigns, improve customer retention, increase customer satisfaction, and maximize marketing return on investment.
