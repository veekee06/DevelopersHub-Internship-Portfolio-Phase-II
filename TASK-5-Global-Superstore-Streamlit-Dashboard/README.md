# Global Superstore Sales Analysis and Interactive Dashboard

# Project Overview
This project analyzes retail sales data from a multinational superstore and develops an interactive business intelligence dashboard using Streamlit. 
The objective was to explore sales performance, profitability, customer behavior, and regional trends while providing stakeholders with a dynamic reporting solution.

# Key Activities
* Data Cleaning and Preparation
* Exploratory Data Analysis (EDA)
* Sales Performance Analysis
* Profitability Analysis
* Customer Analysis
* Regional Performance Analysis
* KPI Development
* Interactive Dashboard Development using Streamlit
* Business Insights and Recommendations
* Dashboard Features
* Region Filter
* Category Filter
* Sub-Category Filter
* Total Sales KPI
* Profit KPI
* Top Customers Analysis
* Interactive Charts and Visualizations

# Key Findings
* Revenue generation is concentrated among a small group of high-value customers.
* Regional sales performance varies significantly across markets.
* Profitability differs across products and transactions, indicating opportunities for optimization.
* Interactive reporting enhances business visibility and supports faster decision-making.

# Business Value
The dashboard enables business stakeholders to monitor performance in real time, identify growth opportunities, improve customer management strategies, and make informed decisions based on key operational metrics.
